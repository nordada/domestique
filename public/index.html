Transmission status check

Drives the header icon's glow with what your torrents are actually doing, not just whether Transmission is reachable: green when everything's idle/seeding with no errors, amber while something's downloading, red if any torrent has a tracker or local error. Unrelated to the webhook Transmission already sends this app on torrent completion.

External indexer

Bookmarks the race indexer/tracker site this instance sources torrents from. Unlike the other header icons, clicking it opens the site itself in a new tab instead of jumping here, and it shows that site's own favicon with an accent ring once a URL is set. The glow layers on top of that once reachability is known: green while the site's responding, red if a check finds it down. Checked on its own heartbeat below rather than the general status-poll interval, so a single slow response doesn't flicker the glow.
